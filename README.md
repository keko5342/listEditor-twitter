# listEditor-twitter
This program can edit your listuser.

このPythonコードはTwitterのRESTAPIを用いてサーバからリストの情報を取得し、リストに登録されているユーザのツイートから画像だけを取得するアプリケーションのコードです。大学の友人からリストを整理するためにアプリケーションを作ってほしいとの申し出を受け、作成しました。

Environmenet
- Python3.6
- python-twitter(pip)
- oauth2(pip)
- urllib(pip)
- pillow(pip)
- screeninfo(pip)
- twitter(pip)
